# MatrixCalculator

WPF-приложение на C#/.NET: матричный калькулятор с русским и английским интерфейсом.

## Возможности

- сложение и вычитание матриц;
- умножение матрицы на матрицу и на число;
- транспонирование;
- определитель;
- обратная матрица методом Жордана-Гаусса;
- режим тестирования на 1-10 вопросов;
- случайные задания по выбранной операции или по всем операциям сразу;
- проверка ответа пользователя;
- пошаговое решение конкретного задания.

## Структура

- `Models/Matrix.cs` - ручная реализация матричных операций через массивы;
- `Services/` - генератор заданий, локализация, форматирование решений;
- `ViewModels/` - MVVM-логика окна, команды и уведомления привязок;
- `MainWindow.xaml` - WPF-интерфейс.

## Запуск

```powershell
dotnet run
```
